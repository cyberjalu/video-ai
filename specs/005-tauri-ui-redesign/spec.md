# Feature Specification: Tauri UI Redesign

## Core User Stories
1. **[US1] App Shell Redesign**: As a desktop user, I want a frameless, native-feeling App Shell with custom drag regions so that the app feels like a modern native application (Corporate Dark theme). (P1)
2. **[US2] Input Mode Refinement**: As a user, I want my input components (InputModeCard) to have glow effects and glassmorphism so they match the modern Corporate Dark aesthetic. (P2)
3. **[US3] View Transitions**: As an interactive user, I want smooth Framer Motion transitions between views (Create, History, Templates) so the app feels responsive. (P2)
